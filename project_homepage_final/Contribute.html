Using tool
1. 자신의 데스크탑에 빈 문서를 만든다.
2. Download Git Repository을 해준다.
3. 구글에서 안드로이드스튜디오 앱을 다운받는다.
4. APTM project 파일을 드래그해 안드로이드
스튜디오 앱에 넣어준다.
5. 안드로이스 스튜디오앱에서 에뮬레이터를 다운받는다.
6. 해당 파일을 실행 시킨다.
7. 에뮬레이터에 사이드바에서 맨 아래 점 세게를 클릭한다.
8. 한양대 에리카를 치고 set location을
누르면 현재 위치 주변 화장실 정보가 뜬다.
